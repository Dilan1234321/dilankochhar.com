# dilankochhar.com

Personal portfolio site — full-stack and AI projects, client work, and contact.

## Stack

- HTML, CSS, JavaScript (no framework)
- [Fraunces](https://fonts.google.com/specimen/Fraunces) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts

## Run locally

Open `index.html` in a browser, or use a simple static server:

```bash
# Python 3
python3 -m http.server 8000

# npx
npx serve .
```

Then visit `http://localhost:8000`.

## Deploy (GitHub Pages)

1. Push to `github.com/dilan1234321/dilankochhar.com`
2. Settings → Pages → Source: Deploy from branch
3. Branch: `main` (or `master`), folder: `/ (root)`
4. Save. Site will be at `https://dilan1234321.github.io/dilankochhar.com` (or your custom domain if configured).
